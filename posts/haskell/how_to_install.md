## Using Stack
1. [Download Stack from here](https://docs.haskellstack.org/en/stable/README/).
2. Do `stack setup`.
3. Done!

## Install GHC only
1. [Download binary package from here](https://www.haskell.org/ghc/) and extract it.
2. There's no step 2.
